Este decodificador usa uma cifra simples de substituição, onde algumas letras minúsculas se transformam em combinações de letras específicas:

a -> ai
e -> enter
i -> imes
o -> ober
u -> ufat

Tecnologias utilizadas:
HTML
JavaScript
CSS
